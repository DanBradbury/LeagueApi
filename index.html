Welcome to LeagueApi

A Ruby Gem which provides powerful and convenient methods to utilize the Riot Developer APIs to their full potential. The gem also provides a 1-1 mapping of all the base methods that Riot provides you with.

To get started working with the gem perform the following.

$ gem install league_api
$ export LOL_KEY="api-key"
#VERIFY INSTALL
$ pry / irb
$ :001 > require 'league_api'
=> true
$ :002 > LeagueApi.available_requests
=> {LeagueApi::Champion=>["get_list", "get_by_id(champion_id)"], LeagueApi::Static=>["get_champion_list", "get_champion_by_name(champ_id)", "get_item_list", "get_item_by_id", "get_mastery_list", "get_mastery_by_id", "get_realm", "get_rune_list", "rune_by_id(id)", "get_summoner_spells", "get_summoner_by_id(id)", "get_versions"], LeagueApi::Game=>["recent_games(summoner_id)"], LeagueApi::Summoner=>["find_by_name(summoner_name)", "find_summoners(summoner_id_list)", "get_masteries(ids)", "get_names(ids)", "get_runes(ids)"], LeagueApi::League=>["by_summoner(id)", "entry_by_summoner(id)", "by_team(team_id)", "entry_by_team(team_id)", "get_challenger(queue=\"RANKED_SOLO_5x5\")"], LeagueApi::Team=>["by_summoner(summoner_id)", "find_teams(team_ids)"], LeagueApi::Stats=>["ranked(summoner_id)", "summary(summoner_id)"]}

If you got the same output as above then things are working and you can feel free to play around with the gem and use the API however you like.
